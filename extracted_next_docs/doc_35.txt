Upgrading: Version 10 | Next.js Menu Using App Router Features available in /app Using Latest Version 15.2.2 Building Your Application Upgrading Version 10 Version 10 There were no breaking changes between versions 9 and 10. To upgrade to version 10, run the following command: Terminal npm i next@10 Terminal yarn add next@10 Terminal pnpm up next@10 Terminal bun add next@10 Good to know: If you are using TypeScript, ensure you also upgrade @types/react and @types/react-dom to their corresponding versions.
